# Trip Holiday - Professional Travel Website

A comprehensive, high-tech travel booking website with professional design and modern effects.

## Features

### Core Pages
1. **Home Page** (`index.html`) - Welcome page with category showcases
2. **Packages** (`packages.html`) - Browse all travel packages with filters and sorting
3. **Booking Pages**
   - Without Flight (`booking-without-flight.html`)
   - With Flight (`booking-with-flight.html`)
4. **Category Pages**
   - Luxury Holiday (`luxury-holiday.html`)
   - International Holiday (`international-holiday.html`)
   - Holiday in India (`india-holiday.html`)
   - Pilgrimage (`pilgrimage.html`)
5. **Blog** (`blog.html`) - Travel tips and stories
6. **Payment** (`payment.html`) - Secure payment gateway with discount coupons
7. **Visa Services** (`visa.html`) - Visa assistance and information
8. **Policy Pages**
   - Cancellation Policy (`cancellation-policy.html`)
   - Date Change Policy (`date-change-policy.html`)
   - Terms & Conditions (`terms.html`)

### Key Features

#### Booking System
- **Two Booking Options:**
  - Booking without flight (domestic packages)
  - Booking with flight (international packages)
- **Comprehensive Forms:**
  - Guest profiles with age verification
  - Adult and children age management
  - Hotel preferences and room categories
  - Car/vehicle selection
  - Sightseeing preferences
  - Meal options
  - Activity selections
  - Complimentary services
  - Special requests

#### Filtering & Sorting
- Filter by package type (Premium, Luxury, Basic)
- Filter by destination (International, India, Pilgrimage)
- Filter by travel type (Family, Couple, Solo, Buddy)
- Filter by budget range
- Sort by: Popular, Low Price, High Price, Duration

#### Payment System
- Multiple payment methods:
  - Credit/Debit Cards
  - UPI Payment
  - Net Banking
  - Digital Wallets
  - EMI Options
  - Pay Later/Partial Payment
- Discount coupon system with available codes
- Secure SSL encrypted payments
- Price breakdown with detailed itemization

#### Design Features
- **Theme Colors:** Orange (#FF7F27) and Black based on logo
- **Professional Effects:**
  - Smooth animations
  - Gradient backgrounds
  - Hover effects
  - Card shadows
  - Slide-in animations
  - Fade-in effects
- **Responsive Design:** Works on all devices
- **Modern UI/UX:** Clean, professional interface
